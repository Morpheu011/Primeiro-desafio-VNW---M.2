Como criar a estrutura básica de um documento HTML linkado ao CSS em 10 passos

  1. 1°- Ligue seu computador(certifique-se de que ele está ligado na tomada), abra o navegador de sua preferência e acesse o Youtube.
  2. 2°- No Youtube, pesquise "Como baixar e instalar o Visual Studio Code" escolha o video de sua preferência e siga os passos.
  3. 3°- Ainda no Youtube, pesquise "Como baixar, instalar e configurar o Git Bash" escolha o video de sua preferência e siga os passos.

    4. Observação: Caso seu sistema operacional for Linux, o terceiro passo não precisa ser executado.

  4. 4°- Else{Caso seu sistema operacional for Linux, no Youtube, pesquise "Como configurar o terminal do Linux no Visual Studio Code" escolha o video de sua preferência e siga os passos.}
  5. 5°- Com o VS Code e o terminal instalados e configurados corretamente, crie uma nova pasta e clique sobre ela com o botão direito do mouse e clique em "Git Bash Here" com o esquerdo.
  6. 6°- Com o terminal aberto digite os seguintes comandos: touch index.html style.css, dê o enter.
  7. 7°- Digite o comando: LS, para ver se os arquivos foram criados com sucesso, caso sim, os titulos dos arquivos irão aparecer.
  8. 8°- Abra o Visual Studio Code digitando o seguinte comando: code .
  9. 9°- Com VS Code aberto, clique no documento index.html que criamos, apenas digite no código: ! . E dê enter, se você fez da forma correta irá aparecer a estrutura básica completa do documento HTML
  10. 10°- Agora iremos linkar o documento HTML ao CSS, para isso só precisamos adicionar o seguinte comando dentro da head (sem os cifrões): <$link rel="stylesheet" href="style.css"$> . Pronto, você deu o primeiro passo para a criação de um documento HTML :) Não se esqueça de mudar a lang(idioma) para o idioma local.
